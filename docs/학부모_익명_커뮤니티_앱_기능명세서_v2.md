# 학부모 익명 교육 커뮤니티 앱 기능 명세서 (PRD) - v2

구글 SSO 기반의 인증된 익명성을 바탕으로, 학부모들이 안심하고 실질적인 학교 생활, 학사일정 및 교육 정보를 공유할 수 있는 플랫폼의 기능 정의서입니다.

---

## 1. 회원가입 및 인증 시스템 (Auth)

| 기능 ID | 기능명 | 기능 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **AUTH-01** | 구글 SSO 연동 | • Google 계정을 통한 간편 로그인/회원가입 지원<br>• 가입 시 구글 고유 UID를 DB에 매핑하여 탈퇴/차단 관리 | **필수 (Must)** |
| **AUTH-02** | 가입 필수 정보 입력 | • 자녀 정보 입력: 학교 검색 및 선택(초/중/고), 현재 학년 설정<br>• 법정동 기반 지역 설정 (GPS 혹은 주소 검색) | **필수 (Must)** |
| **AUTH-03** | 가명 프로필 자동 생성 | • 구글 실명 대신 **[지역명 + 자녀학년 + 랜덤형용사/명사]** 조합으로 닉네임 자동 고정 (예: *반포초5 든든한올빼미*)<br>• 유저가 임의로 실명이나 개인정보를 닉네임에 노출하지 못하도록 제한 | **필수 (Must)** |

---

## 2. 학교별/지역별 커뮤니티 (Community)

| 기능 ID | 기능명 | 기능 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **COMM-01** | 맞춤형 피드 (게시판) | • **[우리 학교 탭]**: 동일 학교 학부모만 진입 가능한 폐쇄형 공간<br>• **[우리 동네 탭]**: 인근 지역(법정동 기준) 학부모들이 소통하는 공간<br>• **[전체 탭]**: 전국 단위 공통 교육/입시 정보 소통 공간 | **필수 (Must)** |
| **COMM-02** | 게시글/댓글 CRUD | • 텍스트 및 이미지 첨부 기능, 익명 투표 기능 지원<br>• '좋아요', '스크랩(저장하기)', '신고' 기능 구현 | **필수 (Must)** |
| **COMM-03** | AI 기반 청정 필터링 | • 상업성 광고, 부동산 선동, 도배성 글 실시간 감지 알고리즘 적용<br>• 비속어 및 특정 개인/학원 저격 비방 글 필터링 및 자동 블라인드 | **필수 (Must)** |
| **COMM-04** | 학부모 소모임 지원 | • "등산", "독서 모임", "카풀" 등 지역 학부모 간 오프라인 소모임 개설 및 참여 기능 | **추후 (Could)** |

## 3. 지식 공유 및 영수증 인증 시스템 (Knowledge & Review)

| 기능 ID | 기능명 | 기능 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **KNW-01** | 질문 & 답변 (Q&A) | • 교육, 입시, 양육 고민 질문 카테고리 운영<br>• 답변 채택 기능 (질문자가 유용한 답변을 채택 시 추가 포인트 지급) | **필수 (Must)** |
| **KNW-02** | **학원 영수증 인증 후기** | • 학원비 결제 영수증 또는 학원 앱 수강 화면 캡처 업로드 기능<br>• **[OCR(문자 인식) 기술 활용]**: 학원명, 결제일자 자동 추출 검증<br>• 관리자 승인 완료 시 '인증된 학부모 후기' 마크 부여 | **권장 (Should)** |
| **KNW-03** | 포인트 마일리지 시스템 | • 글 작성, 답변 채택, 영수증 인증 성공 시 포인트 지급<br>• 1일 마일리지 획득 상한선(어뷰징 방지) 제한 로직 적용 | **필수 (Must)** |

## 4. 핵심 편의 및 학사 연계 기능 (Utility & Calendar)

| 기능 ID | 기능명 | 기능 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **UTIL-01** | **학사일정 연계형 라이브 캘린더** | • 공공데이터포털(NEIS API) 연동을 통한 학교별 학사 일정 자동 동기화<br>• **[일정별 독립 커뮤니티 생성]**: 중간고사, 소풍, 방학 등 각 일정 클릭 시 하단에 **해당 일정 전용 익명 댓글 창** 활성화<br>• **[학년별 필터 적용]**: 댓글 창 상단에 학년별 탭을 제공하여 본인 자녀 학년에 맞는 실시간 댓글만 필터링 기능 | **필수 (Must)** |
| **UTIL-02** | 학사일정 댓글 푸시 알림 | • 유저가 '관심 일정'으로 등록하거나 본인이 댓글을 남긴 학사일정에 새로운 학부모 댓글/대댓글이 달릴 경우 실시간 푸시 알림 발송 | **권장 (Should)** |
| **UTIL-03** | 선배 맘 노하우 PDF 마켓 | • 활동 점수가 높은 우수 회원이 작성한 교육 팁(PDF) 등록 기능<br>• 학사일정 댓글 창 내에 관련 노하우집(예: 중간고사 일정 댓글 창 -> '우리 학교 수학 기출 경향 분석 PDF')을 연계하여 노출 및 소액/포인트 결제 유도 | **추후 (Could)** |
| **UTIL-04** | AI 자녀 양육/입시 멘토 | • 24시간 실시간 AI 상담 챗봇 피드<br>• 사춘기 자녀 대화법, 모의고사 성적 기반 대학 라인 가이드 등 기초 상담 제공 | **추후 (Could)** |

## 5. 관리자 시스템 (Back-Office)

| 기능 ID | 기능명 | 기능 상세 설명 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **MNG-01** | 회원 및 권한 관리 | • 불량 유저 구글 UID 기준 영구 차단(Ban) 및 임시 정지 기능<br>• 자녀 학년 수동 변경 및 인증 처리 권한 | **필수 (Must)** |
| **MNG-02** | 콘텐츠 및 신고 관리 | • AI가 1차 필터링한 위험 게시글 모니터링 및 복구/삭제<br>• 유저 신고 누적 글 실시간 대시보드 처리 | **필수 (Must)** |
| **MNG-03** | 영수증 검증 대시보드 | • 유저가 제출한 학원 영수증 이미지와 OCR 추출 값을 관리자가 최종 승인/반려하는 검증 툴 | **권장 (Should)** |
| **MNG-04** | **학사일정 댓글 모니터링** | • 시험 기간, 소풍 기간 등 특정 학사일정에 트래픽이 몰릴 때 해당 댓글 창의 악성 저격, 허위 사실 유포, 광고성 도배 글을 집중 모니터링하고 관리할 수 있는 전용 모니터링 뷰 | **필수 (Must)** |

## 💡 개발 단계 제안 (Roadmap)

1. **Phase 1 (MVP - 약 2~3개월):** 
   * `AUTH-01~03`(구글 로그인 및 가명 설정) + `COMM-01~02`(기본 커뮤니티) + `KNW-01`(Q&A) 중심의 기본 게시판 형태로 오픈하여 유저 반응과 활성화 수준을 검증합니다.
2. **Phase 2 (차별화 - 오픈 후 1~2개월):** 
   * 핵심 킬러 서비스인 `KNW-02`(영수증 인증)와 `UTIL-01`(학사 일정 및 댓글 연동)을 고도화하여 기존 맘카페 유저를 강력하게 락인(Lock-in) 시킵니다.
3. **Phase 3 (수익화 - 안정기):** 
   * `UTIL-03`(PDF 노하우 마켓)과 지역 학원 타겟팅 광고 시스템을 붙여 비즈니스 모델을 본격 가동합니다.
